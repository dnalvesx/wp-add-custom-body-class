== Description ==

The "Add Custom Class on Body" plugin allows you to dynamically add custom CSS classes to the `<body>` element of your WordPress site. You can customize the classes based on the current URL or the post's slug.

= Features =

- Adds a dynamic CSS class to the `<body>` element.
- Customizes the class based on the post's slug or the current URL.
- Enhances the styling possibilities for your WordPress site.

== Installation ==

1. Upload the `add-custom-class-on-body` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Usage ==

Once the plugin is activated, it will automatically add a custom CSS class to the `<body>` element based on the post's slug or the current URL.

== Frequently Asked Questions ==

= Can I customize the CSS class added to the body element? =

Yes, you can customize the CSS class added to the body element by modifying the plugin settings.

= How can I remove the CSS class added by this plugin? =

To remove the CSS class, simply deactivate the plugin.
